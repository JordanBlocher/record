record by certik

Records the screen and audio.

The record.py script uses gtk's "get_from_drawable()" function to take screenshots and modified alsa's arecord to record the sound.

Install

See the README file.

Demo

To embed the video to your pages, use the theora2flv script to convert the theora video to a FLV file. Then upload the file somewhere to the internet and use the following code: 
<a href="http://sage.math.washington.edu/home/ondrej/scratch/video/v.ogv.flv"
    style="display:block;width:667px;height:488px" id="player"> </a>
<script src="http://femhub.googlecode.com/svn/trunk/swf/flowplayer-3.1.1.min.js"
    type="text/javascript"> </script>
<script type="text/javascript">
flowplayer("player", "http://femhub.googlecode.com/svn/trunk/swf/flowplayer-3.1.1.swf");
</script>
Just edit the href to point to your FLV video and then adjust width (= width of your video) and height (= height of your video + 24 that is occupied by the bottom toolbar).
